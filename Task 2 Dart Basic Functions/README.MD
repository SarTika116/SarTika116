# ✨ Project Function Dart - Sartika

Halo! 👋
Ini adalah project sederhana yang aku buat untuk belajar **function di Dart**.

Di project ini aku mencoba memahami 4 jenis function:

* Function tanpa parameter tanpa return
* Function tanpa parameter dengan return
* Function dengan parameter tanpa return
* Function dengan parameter dan return

---

# 📚 Isi Project

Project ini terdiri dari 3 topik utama:

## 1. Menyapa 👋

Berisi function untuk menampilkan pesan sapaan.

Contoh:

* Menampilkan "Halo"
* Menyapa dengan nama
* Mengembalikan string sapaan

---

## 2. Kalkulator Luas Persegi 🔲

Menghitung luas persegi dengan rumus:

Luas = sisi × sisi

Function dibuat dalam berbagai variasi:

* Tanpa input
* Dengan input sisi
* Dengan return value

---

## 3. Kalkulator Diskon 💸

Menghitung harga setelah diskon.

Rumus:
Harga akhir = harga - (harga × diskon / 100)

Function juga dibuat dalam 4 tipe berbeda.

---

# ⚙️ Penjelasan Teknis

Setiap topik memiliki 4 jenis function:

1. Tanpa parameter & tanpa return
   = langsung print hasil

2. Tanpa parameter & dengan return
   = mengembalikan nilai

3. Dengan parameter & tanpa return
   = menerima input, lalu print hasil

4. Dengan parameter & dengan return
   = paling fleksibel, bisa digunakan kembali

---

# 🚀 Cara Menjalankan

1. Pastikan Dart sudah terinstall
2. Jalankan perintah:

```
dart main.dart
```
